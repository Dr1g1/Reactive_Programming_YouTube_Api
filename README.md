# Task 4:  
Using the principles of Reactive Programming and the YouTube API, implement an application for analyzing comments for given videos (it is possible to provide video identifiers). For the collected comments, determine the proportion of likes and dislikes relative to the total number of reactions on a per-video basis. Display the obtained results.  

Added modification: Using the YouTube API, we noticed that the number of dislikes is no longer visible. Therefore, instead of the number of dislikes, we added the number of comments, since this value is available using the YouTube API.
